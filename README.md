# Modelo-Anat-mico-Desmontable-del-Cerebro-en-3D
Diseño y fabricación de un modelo anatómico del cerebro humano mediante impresión 3D, estructurado en secciones desmontables que permiten identificar y analizar sus diferentes partes. Este recurso facilita el aprendizaje interactivo y la comprensión de la anatomía cerebral.
